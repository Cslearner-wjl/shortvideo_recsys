# 技术设计: 单视频播放模式与浅色风格回退

## 技术方案
### 核心技术
- Vue 3 + Vite + TypeScript
- Element Plus（管理端）

### 实现要点
- 推荐/榜单页改为单视频渲染：维护当前索引，显示单条数据
- 使用 `<video>` 标签直接播放视频，切换时自动 `load()` + `play()`
- 监听键盘上下键事件，切换索引并播放
- 恢复浅色主题变量与组件样式
- 全量中文文案替换（导航、按钮、空态、统计标签）

## 架构决策 ADR
### ADR-003: 单视频切换与自动播放策略
**上下文:** 用户希望单视频沉浸式展示并支持键盘切换
**决策:** 使用列表数据 + 当前索引渲染单条视频，切换时触发 `<video>` 自动播放
**理由:** 改动范围可控，兼容现有接口与分页逻辑
**替代方案:** 全屏路由级重构 → 拒绝原因: 改动范围过大
**影响:** 需处理自动播放浏览器限制与焦点控制

## 安全与性能
- **安全:** 键盘事件仅在页面挂载时监听，卸载时移除
- **性能:** 单视频渲染降低列表压力；切换时按需加载
