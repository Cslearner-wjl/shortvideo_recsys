# 技术设计: 互动对接与持久化校验

## 技术方案
### 核心技术
- Vue 3 / Vite / TypeScript
- Spring Boot / MyBatis-Plus / MySQL

### 实现要点
- 前端统一使用 `request` 携带 Token；写操作前检查登录态，未登录提示并中止。
- 互动操作（评论/点赞/收藏）在 Feed/Hot/VideoDetail 统一走后端接口，成功后同步状态与计数。
- 401/403 等鉴权错误统一处理：提示登录或重新登录。
- 手工验收增加数据库验证步骤（comment_likes/video_likes/video_favorites/comments）。

## 安全与性能
- **安全:** 写操作必须登录；接口返回 401 时不再重试并提示。
- **性能:** 保持原有懒加载与分页，不引入额外重查询。

## 测试与部署
- **测试:** 手工回归 + MySQL 数据校验（插入/删除关系表与评论表）。
- **部署:** 前端代码更新，后端无需改动（若发现接口异常再补修）。
