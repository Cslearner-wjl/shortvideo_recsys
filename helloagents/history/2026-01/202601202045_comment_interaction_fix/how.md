# 技术设计: 评论交互修复与布局调整

## 技术方案
### 核心技术
- Vue 3 / Vite / TypeScript

### 实现要点
- Feed/Hot 页面使用 CommentBox 显示评论输入与发送按钮。
- 修复热门页面按钮可用性（移除遮挡层/改用可点击按钮结构）。
- 调整详情按钮在互动栏中的位置，位于点赞按钮上方。

## 安全与性能
- **安全:** 无新增输入通道，仅恢复评论输入。
- **性能:** 仅前端布局与交互修复，无性能影响。

## 测试与部署
- **测试:** 手工回归：Feed/Hot 发表评论、详情按钮位置、互动按钮点击。
- **部署:** 仅前端代码变更。
