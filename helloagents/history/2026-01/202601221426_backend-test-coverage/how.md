# 技术设计: 后端补测与报告完善

## 技术方案
### 核心技术
- Spring Boot Test + MockMvc
- H2 内存数据库（test profile）
- Mockito（必要时使用 @MockBean 隔离外部依赖）

### 实现要点
- 权限/参数校验类测试：验证未授权访问、非法参数、空参数的错误码与状态码
- 推荐/热榜边界测试：覆盖 page/pageSize/游标非法输入与裁剪逻辑
- 互动异常与幂等测试：验证重复点赞/取消的幂等性与未审核资源的 404 处理
- 测试数据隔离：统一使用唯一后缀构造用户/视频，避免冲突

## 安全与性能
- **安全:** 不引入真实凭据；仅使用测试账号与随机数据
- **性能:** 保持测试规模可控，避免一次性生成超大数据集

## 测试与部署
- **测试:** `mvn -pl backend test`
- **环境:** 默认使用 test profile；Minio 相关集成测试保持原有条件开关
