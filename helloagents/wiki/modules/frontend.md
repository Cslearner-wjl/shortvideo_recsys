# frontend

## 目的
承载前端相关代码与构建产物的聚合模块目录，包含 `admin/` 与 `user/` 两个子项目。

## 模块概述
- **职责:** 提供管理端与用户端的前端页面与交互能力
- **状态:** 🟡开发中
- **最后更新:** 2026-01-22

## 规范

### 需求: 前端应用结构
**模块:** frontend
本模块为前端聚合目录，具体需求与实现细节分散在子项目中。

#### 场景: 新增/维护前端功能
前置条件: 选择目标子项目（`admin` 或 `user`）
- 子项目独立开发、构建与运行
- 变更需同步更新对应子项目模块文档（如 `frontend-admin` / `frontend-user`）

## API接口
本模块不直接定义后端API，API对接信息以子项目为准。

## 数据模型
本模块不直接定义数据模型。

## 依赖
- backend
