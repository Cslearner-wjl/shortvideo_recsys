# frontend-admin

## 目的

提供管理端后台，用于管理员登录、用户管理、视频审核与运营数据看板。

## 模块概览

- **职责:** 登录鉴权、用户管理、管理员账号管理、视频管理、数据看板
- **状态:** 稳定可用（M2.3 落地）
- **最后更新:** 2026-01-22

---

## 规范（核心场景）

### 需求：管理端登录
支持管理员使用 Basic Auth 登录，成功后进入数据看板。

### 需求：用户管理
支持分页、关键词搜索以及冻结/解封操作，操作后列表刷新。

### 需求：管理员账号管理
支持管理员账号新增、编辑、删除与改密。

### 需求：视频管理
支持视频列表、审核、设热门、删除与上传。
- 列表当前复用 `/api/videos/page`，仅展示审核通过的视频

### 需求：统一浅色主题
管理端登录/用户管理/视频管理/数据看板统一浅色视觉风格与层级，提升可读性与操作聚焦。

### 需求：数据看板
展示播放趋势、用户增长、视频发布量、TopN 热门、活跃度等图表。

---

## 依赖

- Vue 3 / Vite / TypeScript
- Vue Router / Element Plus / ECharts
- backend 管理端 API（Basic Auth）

## 测试

- 单元测试: `npm run test:unit`（Vitest）
- 端到端测试: `npm run test:e2e`（Playwright，默认端口 4174）
- 报告输出: `docs/tests/frontend-admin/`
